# Анализ конкурентов

> Дата: 2026-06-10

## Общий вывод

**Квиз-воронки нет ни у одного конкурента.** Все работают по схеме:
каталог готовых планов → форма заявки → звонок менеджера.

Это наша точка входа: квиз-воронка + персонализированное интервью.

**Ценовой диапазон рынка:**
- Готовый шаблон: 2 000–4 000 ₽
- Индивидуальный план: 6 000–15 000 ₽
- Наш ценовой ориентир (с квизом + интервью): 8 000–15 000 ₽

---

## 1. moybiznesplan.ru — «Мой План»

**Что продают:** 40+ готовых бизнес-планов по нишам.
**Цены:** 3 490–3 990 ₽ (скидка с 8 500–8 950 ₽).
**Воронка:** нет квиза. Каталог с фильтром по нише → оплата → скачивание PDF.

**Сильное:**
- 3 000+ реализованных кейсов, 427 отзывов на Яндекс-картах (5★)
- Лид-магниты: книга «15 ошибок при оформлении СК», конструктор идей, тренажёр теста
- Лицо эксперта (Владислав Марясов, 18 лет опыта)
- YouTube-канал, Telegram, соцсети

**Слабое:**
- Нет квиза и персонализации
- Цена зависит от региона — непрозрачно
- Длинная страница без чёткой иерархии
- Слабые CTA-кнопки

---

## 2. bzplan.ru

**Что продают:** бесплатные PDF-шаблоны + индивидуальный план за 6 000 ₽.
**Воронка:** список категорий → PDF бесплатно, либо форма заказа внизу страницы.

**Сильное:**
- Чёткая цена (6 000 ₽)
- Гарантия: бесплатные доработки до одобрения
- Прямой контакт (телефон, Telegram)

**Слабое:**
- Нет отзывов и кейсов
- Нет фильтрации клиентов
- Минималистичный дизайн — низкое доверие
- Нет объяснения «почему именно у нас одобрят»

---

## 3. fin-dir.com

**Что продают:** шаблон 2 000 ₽, индивидуальный план от 7 000 ₽.
**Воронка:** простая форма заявки → звонок.

**Сильное:**
- Заявляют 95% одобрения
- Отзывы с фото
- Чёткий список услуг

**Слабое:**
- Нет квиза
- CTA слабый («Узнать цену» вместо мотивирующего)
- Нельзя выбрать нишу перед заявкой
- Нет видео-демонстрации

---

## 4. soccontract.com

**Что это:** контент-сайт (блог + инструменты).
**Что есть:** 450 вопросов для теста МСП, калькулятор прибыльности бизнеса, статьи.
**Монетизация:** реклама + услуги.

**Что взять себе:**
- Идея калькулятора прибыльности — хорошо работает как лид-магнит
- База вопросов для теста МСП — можно использовать при подготовке клиента

---

## 5. agro365.ru

**Что продают:** бизнес-план за 4 500 ₽.
**Сайт не отвечал** при анализе — возможно, низкая активность.
